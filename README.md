# install

```
$ pkg update && pkg upgrade
$ pkg install git
$ pkg install python2
$ pip2 install requests
$ pip2 install mechanize
$ pip2 install bs4
$ git clone https://github.com/rahmat4/ripper
$ cd ripper
```

# run

```
$ python2 ripper.py
```
